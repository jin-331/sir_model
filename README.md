# SIR モデルを用いたコロナ感染グラフを Python で作成してみた
最近コロナ関連のニ用いた等で SIR モデルという言葉をよく聞くようになりました。
SIR モデルはインフルエンザの感染率グラフ等にも用いられており、コロナ対策にも利用されています。  
今回は、その SIR モデルを用いて Python でコロナ感染率をグラフ化してみました。  

## SIR モデルとは
SIR モデルとは伝染病の感染の様子を表現する数理モデルです。  
S は未感染者、I は感染者、R は回復者を表しています。
